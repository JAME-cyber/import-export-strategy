# Transcripts — ARCHIVÉS (politique droits d'auteur)

Les 40 transcripts bruts YouTube ont été **déplacés vers `.archive/transcripts-raw/`**
hors du repo distribué, conformément à l'audit juridique (`scripts/audit-juridique-gpt5.md`).

## Pourquoi

Les transcripts bruts sont des reproductions d'œuvres audiovisuelles protégées
(CPI art. L122-4). Leur usage commercial indirect dans le Roster présente un
risque de contrefaçon (L335-2/L335-3).

## Ce qui remplace

La base de connaissances opérationnelle vit dans `knowledge-base/synthesis/`
sous forme de **notes originales non verbatim**, sourcées par URL
(voir `NOTICE-SOURCES.md` à la racine du projet).

## Si vous avez besoin des transcripts en local

Ils restent accessibles dans `.archive/` (hors git). Ne jamais les exposer
dans le produit distribué, la landing, ou un dépôt public.
