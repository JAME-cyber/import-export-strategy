## 1. MARCHÉS CIBLES PRIORITAIRES

### 🇫🇷 France — Marché domestique
- **TVA** : 20% (déductible pour entreprises)
- **Droits douane** : 0-17% selon produit
- **Avantages** : Marché connu, proximité, infrastructure logistique
- **Niches portrices** : Équipement pro, mobilier, textile, électronique
- **Réglementation** : CE obligatoire, normes françaises

### 🇪🇺 Europe — Marché élargi
- **TVA** : Variable (19-27% selon pays)
- **Avantages** : 450M+ consommateurs, marché unique
- **Hot spots** : Allemagne (logistique), Espagne (textile), Italie (design)
- **Distributeur** : Amazon Pan-EU, Cdiscount International

### 🇲🇦🇩🇿🇹🇳 Afrique du Nord — Marché en croissance
- **Opportunité** : Marché immense, peu de concurrence locale
- **Produits demandés** : Textile, électronique, machines, cosmétique
- **Logistique** : EMS Négoce, transitaires spécialisés
- **Points attention** : Paiements sécurisés, réglementations variables
- **Avantage** : Marge très élevée (100-300%)

### 🇸🇳🇨🇮🇨🇲 Afrique Subsaharienne
- **Potentiel** : Énorme, en forte croissance
- **Produits** : Équipement agricole, machines, textile, électronique
- **Logistique** : Maritime principal, transitaires spécialisés
- **Risque** : Paiement, délais, douane complexe

### 🇦🇪 Moyen-Orient
- **Marché** : Dubai = hub régional
- **Avantages** : Pas de TVA élevée (5%), clientèle aisée
- **Produits** : Premium, luxe, high-tech

## 2. ANALYSE SECTORIELLE

### 🏗️ Machines & Équipements industriels
- **Prix usine Chine** vs **Prix revente** : Ratio 3-10x
- **Produits clés** : Compresseurs, mini-pelles, machines chantier
- **Marchés** : Afrique, Moyen-Orient, Europe de l'Est
- **Avantage Chine** : Aucune usine européenne à ce prix/flexibilité
- **Exemple** : Machine industrielle achetée 2000€ → vendue 8000-15000€

### 🪑 Mobilier & Aménagement
- **Hub** : Foshan (Chine) — plus grand marché mondial
- **Produits** : Meubles design, spa, sauna, mobilier outdoor
- **Marge** : 100-300%
- **Container type** : 1×40'HC = ~67m³ de mobilier
- **Investissement** : 15-50K€ par container

### 🏠 Maisons modulaires & Container houses
- **Tendance forte** : Partout dans le monde
- **Prix usine** : 5-30K€ selon taille et finitions
- **Prix revente** : 30-100K€
- **Certifications** : Normes construction locales à vérifier

### 📱 Électronique & Tech
- **Hub** : Shenzhen
- **Marge** : 30-100% selon produit
- **Risques** : Contrefaçon, certifications, batteries lithium
- **Produits porteurs** : Accessoires connectés, IoT, wearables

### 🧴 Cosmétiques (marque blanche)
- **Laboratoires chinois** : Qualité désormais comparable voire supérieure
- **Formules** : Naturelles, certifiées possibles
- **MOQ** : À partir de 1000 pièces
- **Marge** : 70-90%
- **Réglementation** : CPSR, CPNP pour Europe

### 🍽️ Équipement restauration/hôtellerie
- **Produits** : Cuisines pro, matériel frigorifique, aménagement
- **Clientèle** : Restaurants, hôtels, food trucks
- **Marge** : 50-150%
- **Exemples** : Food truck Chine → sur-mesure, container complet

## 4. FENÊTRE D'OPPORTUNITÉ 2025-2026

### Contexte favorable :
- **Guerre commerciale US-Chine** → Fournisseurs cherchent clients européens
- **Prix compétitifs** : Usines font des concessions pour clients EU
- **Flexibilité MOQ** : Plus accessible qu'avant
- **Logistique digitalisée** : Plus facile à gérer à distance
- **IA** : Outils d'automatisation réduisent barrières à l'entrée

### Actions immédiates recommandées :
1. **Sélectionner un secteur** selon expertise et budget
2. **Créer la structure juridique** (SASU ou SAS en France)
3. **Ouvrir compte professionnel** multidevise
4. **Contacter 5-10 fournisseurs** sur 1688 pour comparer
5. **Commander échantillons** pour valider qualité
6. **Première commande test** en LCL (partage container)
7. **Vendre rapidement** pour valider le marché
8. **Réinjecter** dans commande plus grosse (container complet)
