# 💬 APPROCHE LINKEDIN — décrocher 3 à 5 entretiens

> **Objectif** : obtenir **15-20 min d'échange**, pas vendre le Roster.
> **Cible** : importateurs solo, gérants de PME e-commerce, e-commerçants qui importent de Chine.
> **Principe** : offrir de la valeur (analyse gratuite) **avant** de demander. Pas de pitch produit.

---

## 🎯 Qui cibler (segments à forte probabilité de pain)

| Segment | Où les trouver | Pourquoi douleur probable |
|---|---|---|
| **Gérants de marque e-commerce FR** (1-10 employés) | LinkedIn : "e-commerce" + "import" + "marque" | Importent déjà, calculs maison, marges serrées |
| **Importateurs dropshipping→marque** (en transition) | Commentateurs de vidéos StartBusinessWorld, SINOSOURCING | Ont le pain, cherchent à professionaliser |
| **Transitaires / commissionnaires** (angle réseau) | CCI, fédérations transport | Connaissent les clients qui galèrent → referrals |
| **Gérants de boutiques physiques qui importent** | LinkedIn + annuaires CCI | Calculent à la main, peu outillés |

**Éviter** : les pure-players du dropshipping (jamais payé, pas de stock), les « coachs e-commerce » (vendent du rêve, pas des imports).

---

## 📨 Message A — Value-first (taux de réponse le plus élevé)

> Pour un gérant e-commerce / importateur actif. On **offre** une analyse, on **demande** 15 min.

```
Bonjour [Prénom],

Vu votre activité [détail spécifique, ex: "autour des accessoires auto /
votre dernière marque maison"] — j'imagine que le sourcing Chine et le
calcul des coûts complets prennent une place non négligeable.

Je travaille en ce moment sur un outil d'analyse anti-biais pour l'import
(une vidéo YouTube qui annonce 300 % de marge, on calcule la réalité :
souvent 15-25 % net).

Si ça vous dit, je vous propose de prendre 15 min pour vous montrer
l'analyse sur un produit que je connais — et si vous avez un produit en
tête, je vous sors le même calcul en brouillon, offert, sans contrepartie.

Ça vous intéresserait d'échanger ?

[Prénom] — analyse import anti-biais
```

**Pourquoi ça marche** : on ne demande rien, on donne. L'offre « ton produit analysé offert »
est l'hameçon. Le 15 min est facile à accepter.

## 📨 Message B — Très court (pour prospects occupés)

```
Bonjour [Prénom],

Question rapide : sur votre dernier import de Chine, qu'est-ce qui vous
a le plus surpris — un coût caché, un certificat, un délai ?

Je travaille sur un outil d'analyse import qui cartographie exactement
ces angles morts. Je cherche juste 3-4 retours de importateurs réels
pour calibrer — 15 min d'échange, je vous montre l'analyse d'un cas réel
en échange.

Ouvert à un échange la semaine prochaine ?

[Prénom]
```

**Pourquoi ça marche** : la question d'ouverture est **factuelle et passée** (méthode Mom Test).
On demande un *retour*, pas un achat. L'importateur se sent expert consulté, pas prospect.

## 📨 Message C — Via réseau / commentaire YouTube (chaleur supérieure)

> À envoyer à des commentateurs actifs des chaînes StartBusinessWorld, SINOSOURCING, CargoFamily
> (on les a déjà dans `data/selected-videos.txt`).

```
Bonjour [Prénom],

Croisé votre commentaire sous la vidéo [titre] — j'ai bossé sur
exactement ce sujet (anti-biais sourcing Chine). J'ai une analyse du
refroidisseur évaporatif à 20€ qui montre pourquoi la marge "300%"
tombe à 15-25 % en réalité.

Vous importez activement ou vous explorez ? Si vous voulez, 15 min
d'échange, je vous montre le calcul sur un cas concret.

[Prénom]
```

**Pourquoi ça marche** : référence partagée = chaleur. On ne vient pas de nulle part.

## 📊 Tactique de volume & suivi

- **Envoyer 15-20 messages** pour viser **3 à 5 entretiens** (taux de réponse attendu 20-30 %).
- **Personnaliser la ligne 1** systématiquement (le copier-coller pur est détecté).
- **Relancer une fois** à J+4 si pas de réponse — court : « Petit up, toujours ouvert à
  l'échange ? Pas de souci si pas le moment. »
- **Ne pas pitcher le Roster dans le message**. Le mot « agent / IA / LLM » est interdit.
  On parle d'**« analyse import anti-biais »**.

---

## 🚫 Anti-biais : ce qu'on NE fait PAS dans l'approche

| Interdit | Pourquoi |
|---|---|
| Promettre un fournisseur validé | Faux — on identifie, on ne valide pas (SGS le fait) |
| Citer des revenus / marges comme promesse | On cite l'**écart** YouTube vs réalité, pas un gain garanti |
| Dire « je cherche des beta-testeurs » | Sonne comme un piège gratuit ; l'offre « ton produit analysé » est meilleure |
| Mass-DM 100 prospects identiques | Taux de réponse < 5 % + réputation brûlée |
| Pitcher le prix maintenant | Le pricing se teste **après** les entretiens, pas avant |
