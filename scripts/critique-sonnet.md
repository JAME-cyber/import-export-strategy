## 1. 🎯 Positionnement stratégique

- **"Les agents font le travail" = PROMESSE VAPORWARE.** Les agents ne "sourcent" rien — ils régurgitent des transcripts YouTube. Un vrai agent négocie sur 1688, inspecte les usines, teste les échantillons, gère les litiges. *Tes agents = chatbots avec de la doc.*
- **Contradiction "pas un cours" vs vendre un cours 1497€.** Incohérence. Soit système d'exécution, soit formation — pas les deux.
- **Le moat "anti-biais" est fragile.** `FOB × 1.4` copiable en 2h. Avantage temporaire 3-6 mois.

## 2. 💸 Coûts cachés & viabilité financière

- **CAC pour vendre 497-1497€ en cold traffic** : Google Ads 150-300€, YouTube Ads 80-200€, FB Ads B2B 200-400€. Vendre 10 formations/mois = budget pub 2000-4000€/mois minimum. Aucun budget marketing mentionné.
- **Niche canicule = MAUVAIS démonstrateur** : saisonnière (4 mois), efficace jusqu'à 35°C seulement, consomme 15L eau/jour, 847 résultats "refroidisseur évaporatif" sur Amazon.fr.
- **Marge "25-40%" surévaluée** : marketing B2C réel 25-35% (pas 20%), retours électroménager 12-15% (pas 8%), SAV/garantie 2 ans -5% omis, stockage/invendus -3 à -8% omis. **Marge nette réaliste : 15-25%.**

## 3. ⚖️ Risques juridiques & conformité

- **RGPD violation flagrante** : collecte email sans mentions légales, sans politique confidentialité, sans base juridique. Amende 4% CA / 20M€.
- **Responsabilité civile sur conseils conformité** : si client suit les agents et se fait saisir 50K€ en douane → qui paie ? Conseil douanier sans habilitation = exercice illégal de profession réglementée.
- **Publicité mensongère** : "412 operators" (faux compteur animé), "pas encore déclassifié" (produit inexistant), "les agents travaillent" (n'existent pas encore). Risque DGCCRF.

## 4. 📉 Saturation & crédibilité marketing

- **Copie trop évidente de stevetan.com** : vocabulaire identique (operator/manifest/classified), structure de page identique. Paraît me-too. Risque parasitisme commercial.
- **Vaporware teasing dangereux** : "pas encore déclassifié" + compteur animé sur produit inexistant = manipulation. Bad buzz garanti.
- **Promesse technique non crédible** : aucune API 1688 connectée, aucun système de négociation automatique, aucune inspection physique.

## 5. 💣 Top 5 KILL FACTORS

1. **Promesse technique impossible** (80%) — agents ne peuvent pas sourcer/négocier/inspecter physiquement → remboursements + bad buzz
2. **Risque juridique RGPD + conseil illégal** (60%) — amendes avant même le lancement
3. **CAC insoutenable sans budget marketing** (70%) — pas de clients sans 2000-4000€/mois de pub
4. **Copie trop évidente de Steve Tan** (40%) — procès + image me-too
5. **Niche démonstrateur défaillante** (50%) — produit saisonnier, saturé

## 6. 🏁 Verdict par livrable

- **LIVRABLE A (Landing) : NON** — RGPD, faux compteur, promesses intenables, copie trop littérale
- **LIVRABLE B (README) : OUI MAIS** — structure cohérente MAIS promesses irréalistes + contradiction
- **LIVRABLE C (Cours) : PAS ENCORE** — contenu bon MAIS 1497€ sans différenciation claire + contradiction

## 7. ✏️ 5 corrections prioritaires (Sonnet)

1. Conformité immédiate : mentions légales + RGPD, supprimer faux compteur, "en développement"
2. Repositionnement cohérent : choisir système OU formation
3. Démonstrateur crédible : remplacer canicule par non-saisonnier
4. Modèle économique réaliste : budgéter 2000-4000€/mois marketing, ou pivot B2B, ou affiliation
5. Différenciation défendable : calculateur coût complet en ligne, rapport anti-biais mensuel
