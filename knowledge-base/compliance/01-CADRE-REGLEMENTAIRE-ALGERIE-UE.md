# 📋 Cadre réglementaire — Import Algérie → UE

**Date:** 5 juillet 2026
**Sources:** EU DG AGRI Factsheet 2026, Access2Markets EU, AGRINFO, US ITA Country Guide, Règlements UE
**Scope:** Dattes Deglet Nour, Huile d'olive, Cosmétiques naturels

---

## 1. Contexte commercial UE ↔ Algérie

| Indicateur (2025) | Valeur |
|---|---|
| Exports UE → Algérie | 2 905 Mio € |
| Imports UE ← Algérie | 99 Mio € |
| Rang Algérie (fournisseur agri-food UE) | 89ème (0.1%) |
| Top imports UE depuis Algérie | Dattes, huile d'olive, produits de la mer |

**Accord d'association UE-Algérie** (depuis 2005) :
- Produits industriels : droits éliminés
- Produits agricoles : préférences **partielles** via contingents tarifaires saisonniers
- L'huile d'olive est un produit **ultra-sensible** (PAC)

## 2. Dattes Deglet Nour (HS 0804.10)

### A. Tarifs douaniers

| Élément | Détail |
|---|---|
| Droit MFN UE | 0% à ~8% selon saison |
| Préférence Algérie | Accès préférentiel via accord d'association |
| Contingent saisonnier | Droits réduits/nuls certaines périodes |
| Vérification | [Access2Markets](https://trade.ec.europa.eu/access-to-markets/) avec HS code exact |

### B. Exigences SPS (Sanitaires et Phytosanitaires)

1. **Certificat phytosanitaire** (obligatoire)
   - Délivré par l'ONPV algérien
   - Doit accompagner chaque expédition

2. **Respect des LMR** (Limites Maximales de Résidus pesticides)
   - ⚠️ L'Algérie figure dans les rapports RASFF/FFN pour non-conformités
   - Risque : blocage frontière + notification RASFF publique
   - Mitigation : analyse labo agréé avant expédition

3. **Conformité alimentaire**
   - R(CE) 178/2002 (traçabilité) + R(CE) 852/2004 (hygiène)
   - HACCP recommandé (pas obligatoire pour dattes brutes)

### C. Étiquetage (R(UE) 1169/2011 INCO)

- Dénomination, pays d'origine, allergènes
- Date de durabilité (DDM/DLC), identité exploitant
- Langue du pays de vente

### D. Certification Bio (optionnel)

- L'Algérie n'a pas de système bio équivalent UE
- Certification par organisme UE : Ecocert, CertiEurope (~500-1500 €/an + audit)
- Mention "Bio UE" uniquement après certification complète

### E. Documents d'importation

| Document | Rôle |
|---|---|
| Certificat d'origine EUR.1 / EUR-MED | Préférences tarifaires |
| Facture commerciale | Base douanière |
| Liste de colisage | Détail contenu |
| Certificat phytosanitaire | Accès UE |
| B/L maritime ou CMR routier | Transport |

## 4. Cosmétiques naturels (R(EU) 1223/2009) ⚠️ COMPLEXE

> **Ne pas commencer par là.** Coûts de compliance élevés avant la première vente.

### Obligations par produit

| Obligation | Coût estimé |
|---|---|
| **Responsible Person (RP)** dans l'UE | 200-2000 €/an |
| **CPSR** (Cosmetic Product Safety Report) par pharmacien/chimiste | 300-800 € |
| Tests stabilité + compatibilité | 200-500 € |
| Notification **CPNP** | Gratuit (mais nécessite CPSR) |
| Dossier Produit (PIF) — 10 ans | Inclusion dans coût RP |
| **Total minimum par produit** | **~600-2000 €** |

### Étiquetage cosmétique

- Liste INCI complète
- N° de lot, PAO (Period After Opening)
- Précautions d'emploi, nom/adresse RP
- Pays d'origine
- 26 allergènes à déclaration obligatoire

### Spécificités produits algériens

| Produit | Potentiel | Note |
|---|---|---|
| Eaux florales (fleur d'oranger, rose) | Bon | Tombent sous 1223/2009 si positionnés "soin" |
| Savon noir, ghassoul | Bon | Tombent sous 1223/2009 |
| Huile d'argan | Bon | ⚠️ Image associée au Maroc, pas à l'Algérie |

## 5. Paiement et rapatriement des fonds

> ⚠️ **Risque critique #1.** Le dinar algérien (DZD) n'est pas librement convertible.

| Option | Faisabilité | Note |
|---|---|---|
| WorldFirst / Stripe / Wise | ❌ | Ne fonctionnent pas vers/depuis l'Algérie |
| SWIFT bancaire | ✅ | Seule option viable, mais lente (3-7 jours) |
| Lettre de crédit (L/C) | ✅ | Sécurise les 2 parties, coût ~0.1-0.25% du montant |
| Transfert informel | ❌ | Illégal, risque pénal |

**Recommandation :** L/C irrévocable pour le premier ordre, puis SWIFT après relation établie.

## 6. Logistique

| Route | Délai | Mode |
|---|---|---|
| Alger/Oran → Marseille | 2-3 jours | Ferry + routier |
| Alger/Oran → Hambourg | 8-12 jours | Maritime (reefer si besoin) |
| Alger → Lyon/Geneva | 5-7 jours | Ferry + routier |

- **Incoterms recommandés :** CIF pour débuter (l'exportateur gère jusqu'au port UE), FOB quand vous serez à l'aise
- **Transport frigorifique :** Dattes à 0-5°C, cosmétiques à température contrôlée

## 7. Feuille de route recommandée

### Phase 1 — Validation (mois 1-2)
- [ ] Contacter la connexion algérienne → évaluer capacité production + qualité
- [ ] Demander échantillons de dattes Deglet Nour (Grades A et B)
- [ ] Faire analyser les échantillons en labo UE (LMR, qualité microbiologique)
- [ ] Identifier 3-5 acheteurs potentiels en Allemagne/France (LinkedIn, TradeMap)

### Phase 2 — Premier envoi test (mois 3-4)
- [ ] Premier ordre modeste (100-200 kg, Grade A)
- [ ] L/C irrévocable pour sécuriser le paiement
- [ ] Certificats phytosanitaires + origine EUR.1
- [ ] Expédition CIF vers port UE proche du premier client

### Phase 3 — Scaling (mois 5-12)
- [ ] Si test concluant : passer à 1-5 tonnes
- [ ] Certification Bio (Ecocert) si la demande le justifie
- [ ] Private label packaging pour marges premium
- [ ] Transition vers FOB + propre freight forwarder

### Phase 4 — Cosmétiques (mois 12+, si pertinent)
- [ ] Identifier 1-2 produits phares (eaux florales, savons)
- [ ] Budget compliance : 1000-2000 €/produit
- [ ] Déléguer RP à un partenaire UE

---

## 8. Checklist de conformité minimale (premier envoi)

- [ ] Certificat phytosanitaire (ONPV Algérie)
- [ ] Certificat d'origine EUR.1 (douane algérienne)
- [ ] Facture commerciale (avec HS code 0804.10)
- [ ] Analyse laboratoire UE (LMR + microbiologie) — faire sur échantillon avant expédition
- [ ] Étiquetage INCO conforme (langue du marché cible)
- [ ] Assurance transport (CIF couvre jusqu'au port d'arrivée)
- [ ] Numéro EORI (si vous importez en votre nom)
