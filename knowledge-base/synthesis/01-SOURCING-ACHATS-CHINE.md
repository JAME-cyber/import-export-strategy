## 1. PLATEFORMES DE SOURCING (par ordre de priorité)

### 1688.com — LA plateforme B2B incontournable
- **Propriétaire** : Alibaba Group (version domestique chinoise)
- **Usage** : B2B, gros volumes, meilleurs prix
- **Avantages** : Prix usine directs, pas d'intermédiaires
- **Inconvénients** : Tout en chinois (utiliser Google Translate)
- **MOQ** : Généralement 50-500 pièces selon le produit
- **Paiement** : Via agent de sourcing ou Alipay business
- **Tip** : Commander un échantillon via Taobao d'abord, puis commander en gros sur 1688

### Taobao — B2C pour échantillons et tests
- **Usage** : B2C, petites quantités, échantillons
- **Avantages** : Produits tendance, nouveautés, pas de MOQ
- **Idéal pour** : Tester un produit avant commande gros volume
- **Tip** : Chercher par image (upload photo du produit)

### Alibaba.com — Porte d'entrée internationale
- **Usage** : B2B international
- **Avantages** : Interface anglais/français, Trade Assurance
- **Inconvénients** : Prix plus élevés que 1688 (revendeurs mélangés)
- **Alternatives** : Global Sources, Made-in-China

### Autres plateformes importantes

## 2. RÉGIONS SPÉCIALISÉES EN CHINE

### Guangzhou (Canton) — Hub commercial
- **Marchés** : Textile, cosmétique, horlogerie, maroquinerie
- **Quartiers clés** :
  - Zhanxi (électronique)
  - Baima (vêtements sans marque)
  - One Link (jouets, décoration)
  - New Asia (gadgets, accessoires)
- **Hôtels recommandés** : Sofitel, Hilton, Paco Hotel, Asia International
- **Foire de Canton** : 2 fois/an (avril & octobre), 3 phases

### Foshan — Mobilier & ameublement
- Plus grand marché de meubles au monde
- Route commerciale immense avec des centaines de showrooms
- Bon pour : mobilier, spa, saunas, décoration intérieure

### Shenzhen — Électronique & Tech
- **SEG Market** : Plus grand marché électronique au monde
- **Fang Time** : Marché Apple (origine & reconditionné)
- Drones, composants, accessoires high-tech

### Yiwu — Petites marchandises & cadeaux
- Le "supermarché du monde"
- Idéal pour : petits objets, décoration, articles cadeaux

## 3. MÉTHODE DE SOURCING

### Processus recommandé :
1. **Identification produit** → Recherche image ou mot-clé sur 1688
2. **Sélection fournisseurs** → Filtrer par : avis clients ≥ 4.5, nombre commandes, certifications
3. **Contact initial** → Demande de prix, MOQ, délais via messagerie plateforme
4. **Commande échantillon** → Via Taobao (1-5 pièces) pour tester qualité
5. **Négociation gros** → Sur 1688 avec cahier des charges précis
6. **Contrôle qualité** → Inspection avant expédition (AQL)
7. **Logistique** → Via agent/transitaire

### Critères de sélection fournisseur :
- ⭐ Avis clients ≥ 4.5/5
- 📦 Nombre de commandes (indicateur fiabilité)
- 🏭 Usine vs revendeur (préférer usine)
- 📜 Certifications (CE, RoHS, ISO selon produit)
- 💬 Réactivité communication
- 🔄 Historique export vers votre marché

## 4. BUSINESS MODEL : AGENT DE SOURCING

### Concept :
- Trouver des clients (commerçants, entreprises) qui veulent importer
- Sourcer les produits pour eux en Chine
- Facturer une commission de **15-20%** (dégressif selon volume)
- Faible capital stock (le client paie la commande), mais **investissements indirects réels** : temps commercial, qualification fournisseurs, traduction, vérifications, assurance et sécurisation contractuelle

### Avantages :
- Pas de stock à gérer
- Pas de risque financier
- Revenus récurrents si clients fidèles
- Expertise qui se monnaye

### Processus :
1. Prospecter des entreprises (B2B)
2. Récupérer leurs besoins produits (images, specs)
3. Sourcer sur 1688/usines locales
4. Négocier et obtenir meilleurs prix
5. Présenter devis avec marge incluse
6. Gérer logistique via partenaire transitaire

## 5. FOIRE DE CANTON — Guide Complet

### Calendrier (2025/2026) :
- **Phase 1** (avril/oct) : Électronique, hardware, lighting
- **Phase 2** (avril/oct) : Meubles, décoration, jouets
- **Phase 3** (avril/oct) : Textile, chaussures, alimentation

### Conseils pratiques :
- Prendre photos carte de visite + produit ensemble (pour mémoire)
- Ne pas ajouter trop de contacts WeChat d'un coup (risque blocage)
- Viser les stands dans les halls officiels (qualité garantie)
- Négocier en personne = meilleurs prix qu'en ligne
- Se concentrer sur Phase 2 pour les meilleurs deals produits finis

### Inscription :
- Via le site officiel cantonfair.org.cn
- Passeport + photo + formulaire
- Entrée gratuite pour les acheteurs internationaux
