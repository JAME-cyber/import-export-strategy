## 1. CONTEXTE — Les canicules européennes (2023-2025)

### Chiffres clés :
- **2023** : Canicule historique, 47.2°C en Sardinia, +3.2°C moyenne Europe du Sud
- **2024** : Vagues de chaleur répétées juin-septembre, 10 pays >40°C
- **2025** : Tendances identiques, saison chaude rallongée (+30 jours vs 2000)
- **Marché EU climatisation** : +12% par an depuis 2022 (Uniclima)
- **France** : 2ème marché EU climatiseurs mobiles derrière l'Italie

### Impact commercial :
- Les pics de chaleur = pics de vente instantanés (stock out en 48h en 2023)
- La demande dépasse l'offre chaque été
- Les produits sont **saisonniers** mais de plus en plus (mars → octobre)

## 3. TOP 5 PRODUITS NICHE — Analyse détaillée

### 🥇 #1 : REFROIDISSEUR ÉVAPORATIF MOBILE (Evaporative Air Cooler)

**Pourquoi ce produit mérite une analyse prioritaire :**
- Technologie 3-en-1 : ventilateur + humidificateur + refroidisseur
- **Aucune évacuation extérieure** (pas besoin de tuyau = avantage vs clim mobile)
- **Consommation 10x inférieure** à un climatiseur (80W vs 1000W)
- **Pas de gaz réfrigérant** = pas de réglementation F-Gaz
- Prix accessible : €150-400 vs €300-600 pour clim mobile
- Concurrence visible en France **moins structurée** que sur les climatiseurs mobiles classiques (à confirmer par analyse Amazon / marketplaces / distributeurs B2B)

| Donnée | Valeur |
|--------|--------|
| Prix usine Alibaba | $25-80 |
| Coût complet import (×1.5) | €41-132 |
| Prix marché EU | €150-400 |
| **Marge nette estimée** | **35-50%** |
| Marché EU 2024 | ~€800M, croissance +18%/an |
| Saisonnalité | Avril-Octobre (7 mois) |
| MOQ typique | 100-500 pièces |
| Poids unitaire | 5-15kg (LCL possible) |
| Certification | CE, RoHS (simple) |
| Ce qu'en disent nos transcripts | StartBusinessWorld: "climatiseur portatif 155¥→80€" ✅ confirmé |

**⚠️ Anti-biais :**
- Marge nette RÉELLE (après marketing + retours) : 25-40% (pas 60-75%)
- Saisonnier → cash flow tendu hors saison
- Produit encombrant → coûts transport élevés
- CE obligatoire mais certificats simples (pas de gaz)

**📊 3 scénarios (100 unités, MOQ standard) :**

|----------|---------------|-------------|-------------|-------------|
| 🔴 Pire | €13,200 | €15,000 | -€800 (-6%) | 40% |

| 🟢 Optimiste | €13,200 | €30,000 | +€8,800 (29%) | 20% |

**Espérance : (0.4×-800) + (0.4×4500) + (0.2×8800) = +€3,280** ✅ Positif

**🏁 VERDICT : OUI MAIS**
- Commencer avec 100 unités LCL (pas container complet)
- Viser le B2B (restaurants terrasses, hôtels, bureaux) = pas de marketing B2C
- Commander en mars pour livraison mai = prêt avant canicule

## 5. ⚠️ ANTI-BIAIS — Ce qu'on ne vous dit pas

| Biais | Réalité |
|-------|---------|
| "Marché en croissance +18%" | Croissance oui, mais marché saisonnier (5-7 mois/an) |
| "Marge 60-75%" | Marge brute FOB. Nette après tout : 25-45% |
| "Pas de concurrence" | Les grossistes EU achètent déjà sur 1688, ils sont juste plus chers |
| "Stock out garanti" | Oui en canicule, mais impossible de prédire QUAND |
| "Produit éco-responsable" | Le refroidisseur évaporatif consomme de l'eau (10-15L/jour) |
| "CE simple" | Oui pour ventilateurs, NON pour machines à glace (électroménager complet) |

### Timing critique :
- Commander en **février-mars** pour livraison **mai**
- Être en stock **début juin** (début saison)
- Être prêt à **réapprovisionner** en urgence mi-juillet si canicule prévue
- **Écouler le stock** fin septembre (soldes / B2B préparation été suivant)
