## 1. MODES DE TRANSPORT

### Maritime (bateau) — Le standard pour l'import
| Paramètre | Détail |
|-----------|--------|
| **Délai** | 30-45 jours (parfois 2-3 mois) |
| **Coût** | Le plus économique (ex: 2000-4000€/container 20') |
| **Volume min** | 1 CBM (mètre cube) en LCL, 1 container en FCL |
| **Idéal pour** | Gros volumes, produits lourds/encombrants |

#### Types de containers :
- **20' Standard (20'DC)** : 33 CBM, charge max ~22 tonnes
- **40' Standard (40'DC)** : 67 CBM, charge max ~27 tonnes
- **40' High Cube (40'HC)** : 76 CBM, charge max ~27 tonnes
- **LCL (Less than Container Load)** : Partage de container, idéal pour petits volumes

### Aérien — Rapide mais cher
| Paramètre | Détail |
|-----------|--------|
| **Délai** | 4-10 jours |
| **Coût** | 4-8x plus cher que maritime |
| **Poids max** | Selon compagnie, généralement 100-500kg pour express |
| **Idéal pour** | Produits légers, haute valeur, urgences |

### Express (DHL, FedEx, UPS)
- Délai : 3-7 jours
- Idéal pour échantillons et petites commandes
- Portes-à-portes avec dédouanement inclus souvent

### Fret ferroviaire (Chine-Europe)
- Nouvelle route de la Soie
- Délai : 14-20 jours
- Coût intermédiaire entre aérien et maritime
- Bon compromis pour certains produits

## 2. INCOTERMS — Qui paie quoi ?

| Incoterm | Responsabilité acheteur | Risque | Recommandation |
|----------|------------------------|--------|---------------|
| **EXW** (Ex Works) | Transport depuis usine, tout | 🔴 Max | Débutant : éviter |
| **FOB** (Free On Board) | Transport maritime + assurance | 🟡 Moyen | Bon compromis |
| **CIF** (Cost Insurance Freight) | Déchargement + douane destination | 🟢 Réduit | Standard |
| **DDP** (Delivered Duty Paid) | Rien (tout inclus) | 🟢 Min | Idéal débutant |

### DDP = Le choix recommandé pour débuter
- Livraison tout compris : marchandise + transport + douane + TVA
- Prix fixe convenu à l'avance
- Pas de stress logistique
- Idéal pour "colis suspects" (petits colis B2C)

## 3. DOUANE & RÉGLEMENTATION

### Documents obligatoires :
1. **Commercial Invoice** (facture commerciale)
2. **Packing List** (liste de colisage)
3. **Bill of Lading / Airway Bill** (titre de transport)
4. **Certificate of Origin** (certificat d'origine)
5. **HS Code** (code douanier harmonisé — 6-10 chiffres)

### TVA & Droits de douane (France/EU) :
- **TVA** : 20% sur valeur CIF + droits de douane
- **Droits de douane** : Variable selon code HS (0-20%, moyenne ~5-8%)
- **Calcul** : Valeur taxable = Valeur CIF × (1 + droit douane) × TVA

### Certifications produits (marché européen) :
| Type produit | Certifications requises |
|-------------|----------------------|
| Électronique | CE, RoHS, EMC |
| Jouets | EN71, CE |
| Textile | REACH, étiquetage composition |
| Cosmétiques | CPSR, notification CPNP |
| Machines | CE, directives machines |
| Alimentaire | HACCP, IFS, BRC |
| Batteries lithium | MSDS, UN38.3 |

### Pièges douaniers courants :
- ❌ **Sous-valorisation** : Ne jamais réduire la valeur déclarée (risque saisie + amende 5x valeur)
- ❌ **Mauvais HS Code** : Contrôle systématique, sanctions lourdes
- ❌ **Contrefaçon** : Peines pénales + confiscation
- ✅ **Toujours** : Vérifier documents avec transitaire, garder traces

## 4. ASSURANCE CARGO

### Pourquoi obligatoire :
- **6%** des cargaisons bloquées en douane
- **1.5%** des marchandises disparaissent en transport
- **10%+** arrivent avec dommages significatifs

### Coût assurance :
- **0.4% à 0.8%** de la valeur CIF
- Exemple : 50 000€ CIF → assurance 250-400€

### Types d'assurance :
- **Assurance ad valorem** : Remboursement valeur réelle (recommandée)
- **Assurance standard maritime** : Seulement 2€/kg (quasi rien)

### Procédure en cas de sinistre :
1. **Documenter immédiatement** : Photos, vidéos à la réception
2. **Réclamation visible** : Sous 7 jours
3. **Réclamation non-visible** : Sous 21 jours
4. **Délai max réclamation** : 1 an

## 5. RÔLES ACTEURS LOGISTIQUES

| Acteur | Rôle |
|--------|------|
| **Transitaire** | Organise le transport de bout en bout |
| **Commissionnaire** | Intermediaire, achète au transporteur, revend au client |
| **Agent de fret (GSA)** | Représente compagnies aériennes cargo |
| **Courtier en douane** | Gère formalités douanières |
| **Consolidateur** | Regroupe marchandises dans un container partagé |
| **Agent de sourcing** | Trouve fournisseurs + gère logistique départ |

### Chaîne logistique type :
```
Usine Chine → Transport local → Entrepôt consolidation → 
Départ port/aéroport → Transport international → 
Port/aéroport destination → Dédouanement → 
Transport dernier kilomètre → Client final
```

## 6. CONTRÔLE QUALITÉ AVANT EXPÉDITION

### 5 points de vérification critiques :
1. **Échantillon** : Contrôle AQL (Acceptance Quality Level)
2. **Documents** : Vérification codes HS, valeurs, conformité commercial invoice
3. **Emballage** : Robustesse (double/triple cannelure), résistance humidité
4. **Chargement** : Supervision empotage container, répartition poids
5. **Scellé** : Vérification plomb du container + numéro de suivi

### Emballage recommandé :
- Cartons double/triple cannelure selon poids
- Palettes bois traité (international)
- Caisses bois certifiées pour produits fragiles
- Remplissage mousse pour produits sensibles
- Film étirable sécurisation palettes
