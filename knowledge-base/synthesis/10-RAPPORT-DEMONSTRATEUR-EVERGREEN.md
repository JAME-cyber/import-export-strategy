## 5. CONFORMITÉ UE — directives applicables, documents à exiger, qui valide vraiment

Produit non électrique : pas de batterie, pas de prise, pas de moteur.

**Conformité applicable :**

| Domaine | Applicable ? | Commentaire |
|---|---|---|
| Règlement CE 1935/2004 contact alimentaire | Oui | Matériaux destinés au contact alimentaire |
| Règlement CE 2023/2006 GMP | Oui | Bonnes pratiques de fabrication |
| DGCCRF France | Oui | Attentes spécifiques France sur aptitude contact alimentaire |
| Inox contact alimentaire | Oui | Composition matière, migration métaux, résistance corrosion |
| Plastique / PP / silicone couvercles | Oui si couvercles inclus | Migration globale et spécifique |
| REACH | Oui indirectement | Substances préoccupantes, colorants, additifs, emballages |
| RoHS | Non | Non électrique/électronique |
| LVD / EMC | Non | Non électrique |
| Marquage CE | Non | Il ne faut pas marquer CE ce produit au titre contact alimentaire |
| WEEE / DEEE | Non | Pas d’équipement électrique |
| EPR emballages France | Oui | Adhésion filière emballages ménagers |
| EPR textile | Non | Non textile |
| Étiquetage Triman / info-tri | Oui | Emballage destiné consommateur France |

**Documents à exiger du fournisseur :**

1. Déclaration de conformité contact alimentaire UE, spécifique au produit.
2. Rapport de test migration globale et spécifique sur inox.
3. Rapport de test migration sur couvercles PP/silicone.
4. Composition matière inox : idéalement certificat matière ou test XRF indépendant.
5. Déclaration REACH SVHC.
6. Rapport de contrôle corrosion / lave-vaisselle, si revendiqué.
7. Fiche technique produit : dimensions, poids, épaisseur, matériau exact.
8. Traçabilité lot : batch, date production, usine.
9. Photos packaging final sans marquage CE abusif.
10. Notice FR : usage, lavage, restrictions, contact alimentaire.

**Qui valide vraiment :**

- Pour ce produit, il n’y a généralement **pas d’organisme notifié CE** comme pour certains produits réglementés.
- La conformité se démontre par :
  - essais d’un laboratoire accrédité ISO 17025 : SGS, Intertek, Bureau Veritas, TÜV, Eurofins ;
  - revue documentaire par un consultant conformité ou avocat produit ;
  - responsabilité finale portée par l’importateur européen.
- La DGCCRF peut contrôler après mise sur le marché. Elle ne “pré-valide” pas le produit.

Point honnête : ce n’est pas “simple”. C’est moins lourd qu’un produit électrique, mais le contact alimentaire est un vrai périmètre de responsabilité.

## 9. DÉCISION GO / NO-GO / PAS ENCORE — verdict + conditions strictes

**Verdict : PAS ENCORE.**

Le produit est evergreen et importable, mais le modèle économique n’est pas démontré au prix médian. Le scénario de base perd environ **1 531 €** sur 6 mois pour 1 000 sets, sans compter le temps opérateur, les litiges longs, ni une éventuelle baisse de prix concurrentielle.

**Conditions strictes pour passer en GO test :**

1. **FOB cible ≤ ¥40 / set**, packaging inclus, sans dégrader inox ni couvercles.
2. Coût complet rendu France hors TVA ≤ **8,80 € / set**.
3. Test matière indépendant confirmant inox annoncé, idéalement inox 304 sur parties contact.
4. Tests contact alimentaire conformes sur inox et couvercles.
5. Prix de vente Amazon testable à **36,90–39,90 € TTC** avec conversion acceptable.
6. Coût publicitaire cible après lancement ≤ **20 % du CA HT**.
7. Taux de retours objectif ≤ **6 %**.
8. Achat initial limité à **1 000 sets maximum**, pas plus.
9. Inspection AQL obligatoire avant solde fournisseur.
10. Pas de marquage CE sur packaging : ce serait un signal de non-maîtrise réglementaire.

Si ces conditions ne sont pas atteintes, le produit reste vendable mais pas suffisamment robuste pour un primo-importateur.

## 10. VÉRIFICATIONS AVANT COMMANDE — checklist concrète

### Fournisseur

- Identifier 3 fournisseurs minimum sur 1688/Alibaba.
- Demander business license chinoise.
- Vérifier cohérence : adresse usine, photos atelier, capacité production, historique export.
- Demander références export UE, sans les considérer comme preuve suffisante.
- Signer un accord **NNN bilingue chinois/anglais** avant partage marque/packaging.
- Vérifier que le fournisseur fabrique réellement les bols et ne fait pas uniquement du trading.

### Produit

- Commander échantillons de 3 fournisseurs.
- Mesurer :
  - poids réel ;
  - épaisseur inox ;
  - diamètre réel ;
  - qualité bords ;
  - stabilité ;
  - ajustement couvercles.
- Faire test aimant uniquement comme indice, pas comme preuve inox.
- Faire test XRF matière sur échantillon retenu.
- Test lave-vaisselle 10 cycles.
- Test odeur couvercles.
- Test chute packaging 80 cm sur carton export.

### Conformité

- Exiger déclaration de conformité CE 1935/2004 spécifique produit.
- Exiger rapport migration inox.
- Exiger rapport migration couvercles PP/silicone.
- Vérifier date, laboratoire, modèle exact, photos, matériaux, normes citées.
- Refaire au moins un test laboratoire indépendant sur échantillon de production.
- Faire relire dossier par spécialiste conformité contact alimentaire.
- Confirmer absence de marquage CE sur produit/boîte.

### Douane / logistique

- Confirmer code HS avec transitaire : présumé **7323.93**, à valider.
- Confirmer taux de droit réel avant commande.
- Demander cotation DDP uniquement pour comparaison, mais privilégier FOB + transitaire contrôlé.
- Vérifier cartons export : double cannelure, poids carton raisonnable, palettisation.
- Simuler coût FBA exact avec dimensions carton et unité finale.
- Prévoir assurance transport.

### Qualité / inspection

- Définir cahier des charges :
  - inox exact ;
  - épaisseur mini ;
  - poids mini ;
  - tolérance rayures ;
  - ajustement couvercles ;
  - absence bords coupants ;
  - packaging FR.
- Inspection pré-expédition SGS / Bureau Veritas / AsiaInspection :
  - AQL II ;
  - défauts critiques : bords coupants, rouille, matière incorrecte, couvercles déformés ;
  - défauts majeurs : bosses visibles, mauvais ajustement, boîte écrasée.
- Paiement conseillé :
  - 30 % acompte ;
  - 70 % après inspection conforme, avant expédition.

### France / mise sur marché

- Créer compte EPR emballages : CITEO ou équivalent.
- Préparer info-tri / Triman.
- Souscrire RC produit.
- Obtenir EAN GS1 si marque propre.
- Préparer notice FR : usage alimentaire, lavage, restrictions micro-ondes, lave-vaisselle si applicable.
- Vérifier marque déposée EUIPO/INPI.
- Vérifier absence de design breveté ou copie trop proche d’une marque existante.
- Prévoir stock de couvercles supplémentaires : **3–5 %**.
- Mettre en place suivi retours par cause dès les 50 premières commandes.
