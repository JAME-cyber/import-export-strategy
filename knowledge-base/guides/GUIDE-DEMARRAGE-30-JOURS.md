# 🚀 GUIDE DE DÉMARRAGE : Import-Export en 30 Jours

## Semaine 1 : Fondations (Jours 1-7)

### Jour 1-2 : Choix du secteur et du business model
- [ ] Identifier vos compétences et réseau existant
- [ ] Choisir un secteur parmi les opportunités identifiées :
  - 🥇 Machines & équipements (meilleur ratio risque/rentabilité)
  - 🥈 Mobilier & aménagement
  - 🥉 E-commerce marque propre
- [ ] Définir le business model : Agent sourcing / Import B2B / E-commerce
- [ ] Estimer budget initial nécessaire

### Jour 3-4 : Structure juridique
- [ ] Choisir statut juridique (recommandé : SASU ou SAS)
- [ ] Créer société via LegalPlace ou similar (~200-400€)
- [ ] Ouvrir compte bancaire professionnel
- [ ] Commander carte bancaire pro

### Jour 5-7 : Recherche fournisseurs
- [ ] Créer compte 1688.com (utiliser Google Translate)
- [ ] Identifier 10-15 fournisseurs potentiels
- [ ] Filtrer sur : avis ≥ 4.5, volume commandes, certifications
- [ ] Contacter 5-8 fournisseurs sélectionnés
- [ ] Demander : prix, MOQ, délais, échantillons

## Semaine 2 : Validation (Jours 8-14)

### Jour 8-9 : Comparaison et sélection
- [ ] Comparer les réponses fournisseurs
- [ ] Sélectionner 2-3 meilleurs candidats
- [ ] Commander échantillons via Taobao ou express
- [ ] Demander certificats qualité (CE, RoHS, etc.)

### Jour 10-11 : Calcul de rentabilité
- [ ] Calculer le coût d'import complet :
  ```
  Coût total = Prix usine + Transport + Assurance + Douane + TVA + Emballage
  ```
- [ ] Estimer prix de vente marché cible
- [ ] Vérifier marge ≥ 40% (B2B) ou ≥ 60% (B2C)
- [ ] Valider ou ajuster

### Jour 12-14 : Préparation première commande
- [ ] Négocier conditions finales (prix, MOQ, délais)
- [ ] Choisir incoterm (DDP recommandé pour début)
- [ ] Sélectionner transitaire/agent logistique
- [ ] Souscrire assurance cargo (0.4-0.8% valeur CIF)

## Semaine 3 : Lancement (Jours 15-21)

### Jour 15-16 : Commande
- [ ] Passer commande (30% acompte standard)
- [ ] Signer contrat NNN si marque propre
- [ ] Confirmer specs produit + packaging
- [ ] Demander photos production en cours

### Jour 17-19 : Préparation vente
- [ ] Créer support commercial (photos, fiches produit)
- [ ] Lister prospects B2B ou configurer boutique en ligne
- [ ] Préparer matériel marketing
- [ ] Si B2C : configurer Shopify ou marketplace

### Jour 20-21 : Pré-expédition
- [ ] Demander inspection qualité (AQL)
- [ ] Vérifier emballage adapté transport international
- [ ] Confirmer documents expédition (BL, commercial invoice, packing list)
- [ ] Valider code HS et conformité

## Semaine 4 : Réception & Vente (Jours 22-30)

### Jour 22-25 : Transit
- [ ] Suivi expédition en ligne
- [ ] Préparer réception (entrepôt, stockage)
- [ ] Confirmer dédouanement en cours
- [ ] Si problème : contacter transitaire immédiatement

### Jour 26-28 : Réception
- [ ] Inspecter marchandise à réception (photos si dommages)
- [ ] Signaler problèmes dans les 7 jours (assurance)
- [ ] Stocker correctement
- [ ] Mettre à jour inventaire

### Jour 29-30 : Premières ventes
- [ ] Lancer campagne de vente
- [ ] Contacter prospects B2B
- [ ] Activer publicités si e-commerce
- [ ] Collecter feedbacks clients
- [ ] Préparer commande suivante

## 💰 Budget minimum par business model

| Model | Budget min | Budget recommandé |
|-------|-----------|------------------|
| Agent sourcing | 0€ | 500€ (comms) |
| E-commerce marque | 2 000€ | 5 000€ |
| Import B2B (LCL) | 3 000€ | 8 000€ |
| Import B2B (FCL) | 10 000€ | 25 000€ |
| Mobilier/Container | 15 000€ | 40 000€ |
| Machines industriels | 15 000€ | 50 000€ |

## ⚠️ Erreurs fatales à éviter

1. ❌ **Sauter les échantillons** → Toujours tester avant gros volume
2. ❌ **Sous-valoriser en douane** → Amende 5x la valeur + saisie
3. ❌ **Pas d'assurance cargo** → Perte totale en cas de problème
4. ❌ **Pas de contrôle qualité** → Produits non conformes
5. ❌ **Micro-entreprise pour import** → Pas de déduction charges
6. ❌ **Payer 100% à l'avance** → Standard = 30/70
7. ❌ **Oublier la protection IP** → En Chine, qui dépose possède
8. ❌ **Pas de contrat avec fournisseur** → Risque de contrefaçon/copie
9. ❌ **Ignorer les certifications** → Blocage douane + saisie
10. ❌ **Ne pas diversifier fournisseurs** → Dépendance totale
