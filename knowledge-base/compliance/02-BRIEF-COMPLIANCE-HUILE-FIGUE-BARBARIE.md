# ⚖️ Brief compliance — Huile de graine de figue de barbarie

**Produit:** Huile de graine de figue de barbarie (*Opuntia ficus-indica* seed oil / prickly pear seed oil)
**Route:** Algérie → UE (France / Allemagne / Suisse)
**Date:** 5 juillet 2026
**Sources:** Access2Markets EU, Règlements UE, EU DG AGRI, publications scientifiques

---

## 1. Classification douanière

### HS Code

| Élément | Code | Note |
|---|---|---|
| **HS international** | **1520.00** | Autres huiles végétales fixes, fluides ou concrètes, brutes, raffinées ou fractionnées |
| **TARIC UE** | 1520.00.00 | Non différencié au niveau TARIC pour les huiles non listées ailleurs |
| **Alternative possible** | 1515.90 | Autres huiles fixes végétales (si classé comme huile grasse non volatile) |

> ⚠️ L'huile de figue de barbarie n'a pas de HS code dédié. Elle tombe sous le chapitre 15 (huiles et graisses animales/végétales). Vérification obligatoire avec un transitaire avant le premier envoi.

### Classification cosmétique

Si l'huile est positionnée comme **produit de soin** (anti-âge, hydratant, sérum visage), elle tombe sous le **Règlement (CE) 1223/2009** (cosmétique). C'est le cas le plus probable.

Si vendue comme **huile alimentaire**, elle relève du règlement alimentaire (plus simple mais marché différent).

## 3. Réglementation cosmétique UE (1223/2009)

### A. Obligations principales

| Obligation | Description | Délai |
|---|---|---|
| **1. Responsible Person (RP)** | Personne physique/morale dans l'UE, responsable de la conformité | Avant mise sur le marché |
| **2. Dossier produit (PIF)** | Formula, CPSR, étiquetage, allégations, méthodes fabrication | Avant mise sur le marché, tenu 10 ans |
| **3. Notification CPNP** | Enregistrement sur le portail CPNP de la Commission UE | Avant mise sur le marché |
| **4. CPSR** | Cosmetic Product Safety Report, rédigé par un diplômé qualifié | Avant mise sur le marché |
| **5. Étiquetage** | INCI, n° lot, PAO, précautions, RP, origine | Sur chaque produit |

### B. CPSR — Cosmetic Product Safety Report

Le CPSR est le document central. Il comprend :

| Partie | Contenu |
|---|---|
| **Part A (Données sécurité)** | Composition qualitative/quantitative, profil toxicologique, impuretés, stabilité |
| **Part B (Évaluation)** | Évaluation de la sécurité, marges de sécurité, conclusion |

Le CPSR doit être rédigé par un **Safety Assessor** diplômé (pharmacien, chimiste, toxicologue avec qualification reconnue dans l'UE).

### C. Allergènes (26 substances)

L'huile de figue de barbarie **pure** (100% *Opuntia ficus-indica seed oil*) ne contient pas d'allergènes cosmétiques au sens du règlement. Cependant :

| Cas | Obligation |
|---|---|
| Huile pure non parfumée | Pas d'allergène à déclarer |
| Ajout de parfum / huile essentielle | Déclaration obligatoire si >0.001% (rinse-off) ou >0.01% (leave-on) |
| Vitamine E ajoutée (tocophérol) | À déclarer dans la liste INCI |

### D. Allégations (claims)

| Allégation | Conditions |
|---|---|
| "Hydratant" | Preuve d'efficacité (test clinique ou ingrédient reconnu) |
| "Anti-âge" | Preuve requise (étude clinique ou littérature scientifique) |
| "100% naturel" | Aucun ingrédient de synthèse |
| "Bio" | Certification bio UE obligatoire (Ecocert, Cosmébio, NaTrue) |
| "Vegan" | Aucun ingrédient d'origine animale, pas de test sur animaux |

> L'huile de figue de barbarie a des preuves scientifiques solides pour les allégations anti-âge, hydratante et anti-oxydante (vitamine E, phytostérols). La littérature existe, il faut la compiler.

## 4. Certification Bio

### Organismes reconnus par l'UE actifs en Algérie

| Organisme | Base | Statut Algérie | Coût estimé |
|---|---|---|---|
| **Ecocert** | France | ✅ Actif (bureau régional Maghreb) | ~800-1500 €/an + audit |
| **Certisys** | Suisse | ✅ Actif via partenaires | ~1000-1800 €/an |
| **Ctrlab** | Italie | ✅ Actif Maghreb | ~800-1400 €/an |
| **CU Brasil / Control Union** | International | ✅ Actif | ~900-1600 €/an |

### Schéma de certification

```
Producteur algérien (ferme/coopérative)
    ↓ demande de certification
Organisme accrédité UE (Ecocert)
    ↓ audit initial + analyse
Certificat Bio UE (valable 1 an)
    ↓ renouvellement annuel
Maintien de la mention "Bio UE"
```

### Normes applicables

| Norme | Niveau | Note |
|---|---|---|
| **R(EU) 2018/848** | Production bio UE | Règlement cadre |
| **Cosmos / COSMetic Organic Standard** | Cosmétique bio | Standard industrie reconnu |
| **Equivalence bio pays tiers** | Algérie | L'Algérie n'est PAS listée comme équivalent → certification par organisme UE obligatoire |

## 7. Checklist de conformité — ordre chronologique

### Étape 1 : Avant l'import (semaines 1-4)

- [ ] Déterminer la classification douanière exacte (transitaire)
- [ ] Obtenir numéro EORI (si vous importez en votre nom)
- [ ] Faire analyser l'huile brute en UE (HPLC + LMR + métaux lourds)
- [ ] Valider le profil chimique vs concurrence marocaine

### Étape 2 : Compliance cosmétique (semaines 3-8)

- [ ] Identifier et contracter un Safety Assessor (pour le CPSR)
- [ ] Faire réaliser les tests de stabilité et compatibilité
- [ ] Rédiger le PIF (formule, étiquetage, allégations)
- [ ] Désigner le Responsible Person (vous, ou un tiers UE)
- [ ] Effectuer la notification CPNP

### Étape 3 : Certification Bio (mois 2-4, en parallèle)

- [ ] Contacter Ecocert (bureau Maghreb)
- [ ] Planifier l'audit initial de la ferme/coopérative algérienne
- [ ] Compiler les preuves de pratiques biologiques

### Étape 4 : Premier envoi (semaines 6-10)

- [ ] Facture commerciale + EUR.1 + certificat phytosanitaire
- [ ] Transport (CIF recommandé pour premier envoi)
- [ ] Dédouanement UE
- [ ] Réception + contrôle qualité

### Étape 5 : Mise sur le marché (semaines 10-14)

- [ ] Conditionnement en flacons UE
- [ ] Étiquetage conforme INCI
- [ ] Lancement e-commerce / démarchage pharmacies
