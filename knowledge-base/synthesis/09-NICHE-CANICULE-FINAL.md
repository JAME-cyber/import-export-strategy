## 🏆 STRATÉGIE RECOMMANDÉE — "PACK CANICULE"

### Phase 1 (Mai-Juin 2026) : Test marché
- Importer 100 refroidisseurs évaporatifs (investissement ~€3,500)
- + 50 kits brumisation DIY (investissement ~€600)
- + 200 ventilateurs multi-fonctions (investissement ~€4,000)
- **Total investissement Phase 1 : ~€8,100**

### Phase 2 (Juillet-Août 2026) : Scale si validation
- Commander container complet refroidisseurs (500 unités)
- + Ligne brumisation pro pour B2B
- **Investissement Phase 2 : ~€25,000-40,000**

| Canal | Produit | Pourquoi |
|-------|---------|----------|
| **B2B restaurants** | Brumisateur terrasse pro | Clientèle captive, récurrent |
| **B2B hôtels** | Refroidisseur évaporatif (lobby) | Besoin été, budget élevé |

| **B2C marketplace** | Ventilateur multi-fonctions | Tout public, pas saisonnier |
| **B2B événementiel** | Refroidisseur + brumisateur | Festivals, weddings, etc. |
