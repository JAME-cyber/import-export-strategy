# 💰 CALCULATEUR COÛTS D'IMPORT

## Formule complète

```
COÛT TOTAL IMPORT = 
  Prix marchandise (FOB)
  + Transport international
  + Assurance cargo (0.4-0.8% CIF)
  + Droits de douane (% selon code HS)
  + TVA (20% en France)
  + Frais transitaire
  + Transport local destination
  + Divers (emballage, inspection)
```

## Calcul détaillé (exemple)

### Données :
- **Produit** : Climatiseurs portatifs
- **Quantité** : 200 pièces
- **Prix unitaire FOB** : 20€
- **Valeur FOB totale** : 4 000€
- **Transport maritime** : 1 500€ (LCL)
- **Incoterm** : FOB

### Étape 1 : Valeur CIF
```
Valeur CIF = FOB + Transport + Assurance
CIF = 4 000 + 1 500 + (5 500 × 0.6%)
CIF = 4 000 + 1 500 + 33
CIF = 5 533€
```

### Étape 2 : Droits de douane
```
Droits = CIF × Taux douane
Droits = 5 533 × 4% (code HS 8415.10)
Droits = 221€
```

### Étape 3 : TVA
```
Base TVA = CIF + Droits de douane
Base TVA = 5 533 + 221 = 5 754€
TVA = 5 754 × 20% = 1 151€
```

### Étape 4 : Frais annexes
```
Frais transitaire : 300€
Transport local : 200€
Inspection qualité : 200€
Total annexes = 700€
```

### Étape 5 : Coût total
```
Coût total = CIF + Droits + TVA + Annexes
Coût total = 5 533 + 221 + 1 151 + 700
Coût total = 7 605€
```

### Étape 6 : Coût unitaire & marge
```
Coût unitaire = 7 605 / 200 = 38.03€
Prix de vente B2B : 70€ (marge 46%)
Prix de vente B2C : 99€ (marge 62%)
CA potentiel B2B : 14 000€ (bénéfice 6 395€)
CA potentiel B2C : 19 800€ (bénéfice 12 195€)
```

## Calculateur rapide (estimation)

```
COÛT TOTAL ≈ Prix FOB × 1.15 à 1.25 (maritime)
COÛT TOTAL ≈ Prix FOB × 1.30 à 1.50 (aérien)
COÛT TOTAL ≈ Prix FOB × 1.05 à 1.10 (DDP tout inclus)
```

### Ratio rapide pour marge :
```
Prix revente ≥ Coût total × 2 (B2B, marge 50%)
Prix revente ≥ Coût total × 3 (B2C, marge 67%)
Prix revente ≥ Coût total × 5 (Premium, marge 80%)
```
